# A-path-planner-for-dual-propulsion-USV-based-on-improved-A-algorithm
Proposed a path planner for dual-propulsion USV based on improved A* algorithm. The grid method is used to construct the navigation environment model. Then, the artificial potential field (APF) algorithm is applied to exert attractive and repulsive potential fields on the target point and each obstacle, and the force on each grid is incorporated into the evaluation function of the A* algorithm for path search. Subsequently, the Bresenham line algorithm and cubic B-spline curve are used to optimize the path.
